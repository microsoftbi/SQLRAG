# 2026-06-09 修改汇总

今天主要做了以下几类修改：

## 1. 后端接口模块拆分

把原来集中在 `backend/main.py` 里的接口按业务拆成了两个 router 模块。

### 新增文件

#### `backend/API_Vector.py`

负责 VectorDB 相关功能：

- 文档管理
- 文档上传
- 文档分块
- embedding 生成
- VectorIndex 写入
- Vector QA
- 配置管理
- 知识库管理
- 向量索引管理

使用：

```python
router = APIRouter(prefix="/vector", tags=["VectorDB"])
```

原接口路径保持不变，例如：

```text
/vector/documents
/vector/documents/upload
/vector/documents/{document_id}/commit-chunks
/vector/index/create
/vector/index/drop
/vector/qa
```

#### `backend/API_Graph.py`

负责 GraphDB 相关功能：

- `/graph/data`
- `/graph/qa`
- LLM 生成 Graph SQL
- SQL 执行和格式化
- SQL Server Graph MATCH 查询
- graph legacy QA 相关辅助方法

使用：

```python
router = APIRouter(prefix="/graph", tags=["GraphDB"])
```

原接口路径保持不变：

```text
/graph/data
/graph/qa
```

---

## 2. 精简 `backend/main.py`

现在 `backend/main.py` 只保留：

- FastAPI app 初始化
- CORS 配置
- 日志配置
- router 注册
- `/`
- `/story`
- `uvicorn.run`

新增 router 注册：

```python
from API_Vector import router as vector_router
from API_Graph import router as graph_router

app.include_router(vector_router)
app.include_router(graph_router)
```

拆分后执行了语法检查：

```bash
py -m py_compile backend/main.py backend/API_Vector.py backend/API_Graph.py
```

结果：通过，无语法错误。

---

## 3. VectorDB 默认分块参数调整

把 VectorDB 的默认分块参数从：

```text
Chunk Size = 1000
Chunk Overlap = 200
```

改为：

```text
Chunk Size = 500
Chunk Overlap = 100
```

修改文件：

- `backend/.env`
- `backend/.env.example`
- `backend/config.py`
- `frontend/src/views/VectorDB.vue`

这样后端默认配置和前端默认显示保持一致。

注意：后端需要重启后才会读取新的 `.env` 配置。
